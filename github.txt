Repository Name: discover_dart
Description: 
  Seven levels of discovering Dart with Code School.
      
Next steps:
  cd git/dart/discover_dart
  git init
  git status
  git add .
  git commit -m 'working version'
  git remote add origin https://github.com/dzenanr/discover_dart.git
  git push -u origin master

Git with Git Bash

Clone a remote repository into a local repository.

cd git/dart
git clone https://github.com/dzenanr/discover_dart.git
cd discover_dart

Read+Write:
https://github.com/dzenanr/discover_dart.git

Read only:
git://github.com/dzenanr/discover_dart.git

Get the latest changes from the remote repository

git pull
or
git pull origin master

Tagging for versions (spirals)

http://learn.github.com/p/tagging.html

git tag
git tag -a s00 -m 'spiral 00'
git tag

To push tags to the remote repository:

git push --tags
Enter passphrase for key ... :

To remove a tag from the remote repository:

http://nathanhoad.net/how-to-delete-a-remote-git-tag

git tag -d s00
git push origin :refs/tags/s00

---------------------------------------------------------------------

Switching from master to spiral

from master to tag
git checkout s01
from tag to branch
git checkout -b s01a
to use branch
git checkout s01a


Switching from spiral to master

to exclude js:
git checkout discover_dart.dart.js
or 
(see http://scottr.org/presentations/git-in-5-minutes/)
git stash

git checkout master
